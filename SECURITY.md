# Security Policy

We take the security of StockPilot Pro seriously. If you believe you have found a security vulnerability, please report it to us as outlined below.

## Supported Versions

Only the latest release version on the `main` branch receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report security vulnerabilities by emailing **security@stockpilot.pro**.

Please include:
* A detailed description of the vulnerability.
* Steps to reproduce or a proof of concept (PoC).
* Potential impact.

We will acknowledge receipt of your report within 48 hours and work with you to coordinate a responsible disclosure and patch.
